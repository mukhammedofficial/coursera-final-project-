# Code of Conduct

## Our Pledge
We pledge to create a welcoming and respectful environment.

## Our Standards
Be respectful and collaborative.

## Enforcement Responsibilities
Project maintainers are responsible for enforcing the code.

## Scope
This code applies to all project spaces.

## Enforcement
Instances of abusive behavior may be reported.

## Enforcement Guidelines
Warnings or bans may be applied if needed.
